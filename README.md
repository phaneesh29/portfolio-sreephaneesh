# Portfolio Website

This is a portfolio website for Sreephaneesha Kanugovi, built with Next.js, React Three Fiber, Framer Motion, and Tailwind CSS.

## Features

- **3D Interactive Background**: A mesmerizing star field background using Three.js.
- **3D Hero Element**: Animated 3D sphere in the hero section.
- **Smooth Animations**: Framer Motion for smooth scroll animations.
- **Responsive Design**: Fully responsive layout for all devices.
- **Project Showcase**: Highlights key projects with links to GitHub and demos.
- **Contact Form**: Functional UI for contact form (requires backend integration for email sending).

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  **Open in browser**:
    Navigate to [http://localhost:3000](http://localhost:3000).

## Tech Stack

-   **Framework**: Next.js 14
-   **Styling**: Tailwind CSS
-   **Animations**: Framer Motion
-   **3D Graphics**: React Three Fiber, Drei, Three.js
-   **Icons**: Lucide React

## Customization

-   Update resume in `public/resume/`.
-   Modify `src/components/*.tsx` to update content.
